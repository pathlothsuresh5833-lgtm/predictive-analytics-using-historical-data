# Predictive Analytics Using Historical Data

## Project Overview
This project focuses on analyzing historical sales data and forecasting future sales trends using Power BI. Predictive analytics techniques were applied to identify patterns, estimate future performance, and support business decision-making.

## Objectives
- Analyze historical sales data
- Identify sales trends and patterns
- Forecast future sales using time-series forecasting
- Generate actionable business insights
- Support strategic planning through predictive analytics

## Tools Used
- Power BI
- DAX
- Sample Superstore Dataset

## Dashboard Pages

### 1. Historical Sales Analysis
- Total Sales
- Total Profit
- Total Orders
- Historical Sales Trend

### 2. Sales Forecasting Dashboard
- Monthly Sales Trend
- Forecasted Sales
- Confidence Interval Analysis

### 3. Forecast Insights & Recommendations
- Key Findings
- Business Recommendations
- Conclusion
